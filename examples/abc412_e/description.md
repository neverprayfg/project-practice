# AT_abc412_e [ABC412E] LCM Sequence

## 题目描述

对于正整数 $n$，定义 $A_n$ 为 $1, 2, \dots, n$ 的最小公倍数。  
给定正整数 $L, R$，请问数列 $(A_L, A_{L+1}, \dots, A_R)$ 中包含多少种不同的整数？

## 输入格式

输入以如下格式从标准输入读入。

> $L$ $R$

## 输出格式

输出数列 $(A_L, A_{L+1}, \dots, A_R)$ 中不同整数的个数。

## 输入输出样例 #1

### 输入 #1

```
4 12
```

### 输出 #1

```
6
```

## 输入输出样例 #2

### 输入 #2

```
123456789 123456789
```

### 输出 #2

```
1
```

## 输入输出样例 #3

### 输入 #3

```
99999990000000 100000000000000
```

### 输出 #3

```
310458
```

## 说明/提示

## 限制条件

- $1 \leq L \leq R \leq 10^{14}$
- $R - L \leq 10^7$
- $L, R$ 为整数

## 样例解释 1

枚举 $A_4$ 到 $A_{12}$ 如下：  
- $A_4 = 12$
- $A_5 = 60$
- $A_6 = 60$
- $A_7 = 420$
- $A_8 = 840$
- $A_9 = 2520$
- $A_{10} = 2520$
- $A_{11} = 27720$
- $A_{12} = 27720$

因此，$(A_4, A_5, \dots, A_{12})$ 中包含 $6$ 种不同的整数。
